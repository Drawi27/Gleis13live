# GLEIS 13

Unabhängiges Radio-Archiv für elektronische Musik aus Essen.

## Features

- SoundCloud & YouTube Player Integration
- Artist-Verwaltung mit Bio, Foto & Social Links
- Admin-Bereich (Sets, Artists, Info-Text, Ticker verwalten)
- User-System mit Login & Favoriten
- Dark Mode
- Mobile-optimiert mit Bottom Navigation
- Animierter Splash-Screen mit Glow-Effekten
- Keyboard Shortcuts (Leertaste, Pfeiltasten, D, F, Esc)
- Share-Button (Native Share API / Clipboard)

## Setup

Einfach die Dateien auf einen Webserver laden oder GitHub Pages aktivieren.

### GitHub Pages

1. Repository erstellen
2. Settings → Pages → Source: `main` branch → Save
3. Seite ist live unter `https://USERNAME.github.io/REPO-NAME`
